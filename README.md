# Open Jumper - 3D game about jumpstyle dancing

## Description
Open Jumper - 3D game about jumpstyle dancing

## Authors
ZABIDEN - main developer of game

## Controls
Space - Jump
A - Kick left
D - Kick right
W - Shuffle forward
S - Shuffle backward
E - Flip

## Thanks for
GLFW - Nice GL lib for C++
Lodepng - Png loader
GLM - For graphics mathematics

## Building
### Windows
For build you need
* MinGW
* make
* G++
Use the powershell, and write
`
git clone https://github.com/zabidenhtf/openjumper.git
cd openjumper
make PLATFORM=WINDOWS
`

### Linux
Its similar to windows
For build you need
* g++ // it can be preinstalled into your distro
* libglfw3-dev
* make // too
Use the terminal, and write
`
git clone https://github.com/zabidenhtf/openjumper.git
cd openjumper
make PLATFORM=LINUX
`
